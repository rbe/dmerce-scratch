TEXTE & OBJEKTE
Inhalte für Ihre Präsenz !

Innerhalb der Präsenz wie auch innerhalb der Administration können Sie Texte und Informationen Ihrem Angebot hinzufügen, diese pflegen und verwalten. Grundsätzlich legen Sie neue Seiten über die Administration an. Bearbeiten können Sie aktive Seiten entweder durch den Aufruf der Seite und einen Klick auf das Symbol    für den Editiermodus oder durch Auswahl der gewünschten Funktion auf der Administrationsseite.

Wir unterscheiden die Bereiche:

  • Informationsbereich (INFO) 
     Hier stellen Sie für Ihre Besucher beliebige Informationen und Texte zur Verfügung.
  • Objektbereiche (OBJEKT)  
     Hier werden je nach Anwendung spezielle Formulare und Funktionen für spezifische Branchenangebote eingestellt.
    z.B. Immobilien, Versicherungen, Shopartikel, Highlightartikel usw..

Texte & Objekte anlegen:

  • Wählen Sie den Bereich <INFO/OBJEKT> für den Sie Informationen hinzufügen wollen.
  • Im folgenden Fenster werden Sie zur Eingabe Ihrer Informationen aufgefordert
  • Fügen Sie Ihre Informationen ein und klicken Sie auf <anlegen> um den Text hinzuzufügen.
  • Mit dem Klick wird Ihr Text an den Server übersandt, in die Datenbank eingetragen und der gewünschten Navigation zugeordnet.
  • Im folgenden werden die Fenster automatisch aktualisiert (refresh). 

Texte & Objekte bearbeiten:

  • Wählen Sie den zu bearbeitenden Text über das Menü an und wechseln Sie mit einem Klick auf das Symbol    in den Editiermodus. Führen Sie hier Ihre Änderungen durch. 
  • Wählen Sie den zu bearbeitenden Text über die Administrationsseite aus, indem Sie die Funktion auswählen..
  • Wählen Sie im folgenden Fenster aus der Liste.
  • Fügen Sie Ihre Änderungen ein und klicken Sie auf <anlegen> um den Text hinzuzufügen.
  • Die Daten werden an den Server übermittelt und das Fenster wird automatisch aktualisiert.
  • Im Anschluß an die Aktualisierung steht Ihnen der Text zur weiteren Bearbeitung zur Verfügung.
  • Möchten Sie den Bearbeitungsmodus verlassen, klicken Sie bitte auf <zurück> oder auf einen beliebigen Menüpunkt.

Texte & Objekte freistellen:
Freistellen bedeutet bestehende Texte von einer Navigation zu lösen.

  • Texte werden nicht gelöscht
  • Texte stehen für eine neue Zuordnung zu anderen Naviagtion zur Verfügung
  • Texte verbleiben für eine spätere Verwendung in der Datenbank
  • Wählen Sie den Bereich (INFO/OBJEKT) aus dem Sie einen Text freistellen wollen.
  • Wählen Sie im folgenden Fenster aus der Liste den freizustellenden Text.

Texte & Objekte löschen:

  • Wählen Sie den Bereich (INFO/OBJEKT) aus dem Sie einen Text löschen wollen.
  • Wählen Sie im folgenden Fenster aus der Liste den zu löschenden Text.
  • Bestätigen Sie das Löschen

Freie Texte & Objekte zuordnen/löschen:
Dei folgenden Aktionen beziehen sich nur auf freigestellte Texte

  • Wählen Sie den Bereich (INFO/OBJEKT) aus dem Sie einen Text zuordnen/löschen wollen.
  • Wählen Sie im folgenden Fenster aus der Liste den gewünschten Texte und die Aktion.
  • Löschen: Bestätigen Sie den Löschvorgang.
  • Zuordnen: Wählen Sie im folgenden Fenster das Menü für die Zuordnung und bestätigen Sie Ihre Auswahl über

Highlights bearbeiten:
Die Highlightfunktion steht Ihnen nur bei "Objekten" zur Verfügung. 
Gekennzeichnete Highlightartikel werden auf der Startseite in der Folge der vergebenen Nummerierung angezeigt. 

  • Wählen Sie "Highlightliste bearbeiten" 
  • Wählen Sie Im folgenden Fenster den zu bearbeiten Artikel oder fügen Sie einen neuen hinzu.
  • Vergeben Sie eine Nummer.
