NAVIGATION
Aufbau und Struktur !

 ACHTUNG:
Wir empfehlen Ihnen bereits im Vorfeld eine Struktur Ihrer späteren Navigation anzulegen.
Sie ersparen sich bei einer späteren Korrektur viel Arbeit.

Grundsätzliches:

  • Eine Navigation oder Subnavigation wird nur angezeigt wenn sie "aktiv" ist und ihr ein "aktiver" Text zugeordnet ist.
  • Navigation oder Subnavigation können auf "Vorrat" angelegt und zu einem späteren Zeitpunkt aktiviert werden.
  • Die Sortierung der Navigation erfolgt nach der Reihenfolge der Anlage.
  • Bei der Anlage einer Navigation bzw. Subnavigation werden Sie aufgefordert einen Text zu definieren. 
    Dieser Text wird bei dem Aufruf des Menüpunktes angezeigt. 
  • Erstellen Sie weitere Texte und ordnen Sie diese einem bestehenden Navigationspunkt zu, werden sämtliche zugeordneten Texte am unteren Bildschirmrand in einer Liste angezeigt. Nun haben Sie die Möglichkeit den Begrüßungstext der Navigation neu zu definieren.
  • Navigationen oder Subnavigationen können nur gelöscht werden wenn ihnen keine Texte zugeordnet sind.

Anlegen einer Navigation/Subnavigation:

  • Wählen Sie den Bereich (INFO/OBJEKT) für den Sie einen Navigationspunkt erstellen wollen.
  • Im folgenden Fenster werden Sie zur Eingabe der Menübezeichnung aufgefordert. Geben Sie den Text ein.
  • Im folgenden Fenster werden Sie aufgefordert zu wählen, ob Sie der Navigation eine Subnavigation (Untermenü) oder einen Text hinzufügen wollen.
  • Wählen Sie <Text anlegen>, geben Sie bitte im folgenden Fenster Ihre Informationen ein.
  • Wählen Sie <Subnavigation>, geben Sie bitte im folgenden Fenster den Namen der Subnavigation ein, quittieren Sie Ihre Eingabe und klicken Sie im neuen Fenster auf <Text anlegen>. Nun öffnet sich ein neues Fenster für die Eingabe Ihre Informationen.
  • ACHTUNG: Navigationen und Subnavigationen werden nur angezeigt wenn aktive Texte hinterlegt sind.
  • Ein Übersicht über Ihre bereits angelegten Navigationen erhalten Sie entweder über <Navigation/Subnavigation bearbeiten> oder über <Sitemap/Übersicht>

Bearbeiten einer Navigation:

  • Wählen Sie den Bereich Navigation bzw. Subnavigation für den Sie einen Navigationspunkt bearbeiten wollen.
  • Wählen Sie einen Eintrag aus dem Bereich (INFO /OBJEKT). 
  • Wählen Sie aus der folgenden Liste den zu bearbeitenden Punkt aus.
  • Führen Sie Ihre Änderungen durch.
  • Quittieren Sie Ihre Eingabe.
  • Die Seiten werden automatisch aktualisiert (refresh).

Löschen einer Navigation:

  • Wählen Sie den Bereich Navigation bzw. Subnavigation aus dem Sie einen Navigationspunkt löschen wollen.
  • Wählen Sie einen Eintrag aus dem Bereich (INFO /OBJEKT). 
  • Wählen Sie aus der folgenden Liste den zu löschenden Punkt aus.
  • ACHTUNG: Navigationen und Subnavigationen können nur gelöscht werden, werden ihnen keine Texte oder Menüs zugeordnet sind.
  • Bestätigen Sie den Löschvorgang.
