LAYOUT
Wählen Sie aus den Vorlagen !

Administration:
Mit Aufruf der Administration finden Sie den Funktionsblock "Layout".

Ändern und Anpassen:
Klicken Sie auf den Eintrag um aus der folgenden Liste ein Layout auszuwählen.
Das gewählte Layout wird automatisch übernommen.

Firmenlogo ändern:
Hier haben Sie die Möglichkeit ein Logo auf den Server zu laden. Dieses Logo wird in der Originalgröße innerhalb der Kopfzeile angezeigt.
Alternativ zum hochladen eines Logos können Sie im Fenster einen Text eingeben, die Schriftart sowie die  Schriftgröße wählen. Ebenso wie das Logo wird der eingegebene Schriftzug innerhalb der Kopfzeile automatisch angezeigt.

Eigenes Layout erstellen:
Um das Layout auf Ihre Wünsche anzupassen laden Sie bitte zunächst über den Link <Download> die Datei <Beispiellayout.zip> auf Ihren Rechner. Wählen Sie hierzu im Fenster Dateidownload "Datei auf Datenträger speichern" und OK. Die angegebene Datei wird nun in das von Ihnen angegebene Verzeichnis kopiert. Öffnen Sie die ZIP-Datei und kopieren Sie die Bilddateien in Ihr Grafikprogramm.
Nun können Sie die Bilddateien beliebig bearbeiten und über die zuvor gewählte Funktion <Eigenes Layout erstellen> auf den Server hochladen. Ihre Dateien werden Ihr persönliches Verzeichnis geladen und stehen Ihnen anschließend zur Auswahl über <ändern und anpassen> zur Verfügung.

WWW öffnen / schließen:
Mit dieser Funktion können Sie während der Umbaumaßnahmen Ihr Web für Aussenstehende vorübergehend schließen.
Klicken Sie auf die Funktion und stellen Sie im folgenden Fenster den "Vorhang" ein.
